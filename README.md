# 💰 Expense Analyzer - Frontend

A modern and responsive Expense Tracker frontend built using React.js and Tailwind CSS. This application helps users manage their daily expenses with authentication, dashboards, charts, and transaction management.

## 🚀 Features

- 🔐 User Authentication (Login & Register)
- 📊 Dashboard with Expense Overview
- 💳 Add, Edit & Delete Expenses
- 📈 Monthly Expense Bar Chart
- 🥧 Category-wise Pie Chart
- 🔍 Search & Filter Expenses
- 📱 Responsive Design
- 🎨 Modern UI with Tailwind CSS

## 🛠️ Tech Stack

- React.js
- Tailwind CSS
- React Router
- Axios
- Chart.js / Recharts (whichever you're using)
- JavaScript (ES6+)

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/amrit1569/expense--analyzer--frontend.git
```

Go to project directory

```bash
cd expense--analyzer--frontend/frontend
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm start
```

The application will run on:

```
http://localhost:3000
```
